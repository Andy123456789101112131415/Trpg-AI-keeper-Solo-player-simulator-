# 克苏鲁的呼唤跑团游戏需求文档

## 1. 游戏概述

### 1.1 游戏名称
克苏鲁的呼唤：守秘人之旅

### 1.2 游戏描述
一款基于《克苏鲁的呼唤》第七版规则的网页跑团游戏，玩家扮演调查员，在守秘人（KP）的引导下展开充满未知与恐怖的冒险。游戏严格遵循COC7版规则，通过属性检定、技能判定和理智值管理推动剧情发展。

## 2. 游戏玩法

### 2.1 核心规则
- 所有游戏事件通过属性值和技能检定来裁决
- 严格遵循《克苏鲁的呼唤》第七版规则书
- 玩家负责投掷骰子（百分骰），系统根据属性/技能值、难度等级和投掷结果判定成功与否
- 所有神秘生物均来自官方规则书，具有明确的战斗数据和生命值

### 2.2 游戏流程
1. **角色创建**：引导玩家按照第七版规则完成角色创建，包括属性分配、技能选择、背景设定
2. **剧本选择**：提供多个剧本简介供玩家选择，每个剧本具有独特氛围和挑战
3. **冒险进行**：系统扮演KP角色，描述场景、扮演NPC、推动剧情发展
4. **检定判断**：玩家投掷骰子，系统根据规则判定结果并描述后果
5. **角色成长**：跑团结束后，根据剧情发展和表现指导角色属性与技能成长
6. **角色死亡处理**：角色死亡后重新创建新角色

### 2.3 操作方式
- 玩家通过文字输入进行行动选择和对话
- 玩家点击按钮进行骰子投掷
- 系统自动记录并显示角色当前属性、技能状态、理智值、生命值和物品清单
- 系统根据投掷结果和规则自动计算检定成功与否

## 3. 游戏功能

### 3.1 角色管理系统
- 角色创建向导（遵循COC7版规则）
- 角色属性面板（实时显示所有属性、技能、理智值、生命值）
- 物品清单管理
- 角色成长记录

### 3.2 剧本系统
- 多剧本选择界面
- 剧本进度保存
- 场景描述展示

### 3.3 检定系统
- 骰子投掷功能（百分骰）
- 自动检定判断（根据属性值、技能值和难度）
- 检定结果反馈

### 3.4 战斗系统
- 生物数据展示（生命值、属性等）
- 战斗回合管理
- 伤害计算

### 3.5 记录系统
- 自动记录角色所有属性变化
- 物品获取与消耗记录
- 剧情关键节点保存

## 4. 设计风格
- **配色方案**：深墨绿色为主色调，搭配暗金色和深褐色，营造神秘压抑的克苏鲁氛围
- **视觉细节**：采用羊皮纸质感背景，手写字体风格标题，边框使用古籍装饰纹样，按钮带有轻微浮雕效果
- **整体布局**：采用卡片式布局，左侧固定显示角色属性面板，右侧为主要剧情展示区域，底部为操作按钮区
- **图标风格**：使用线条勾勒的克苏鲁元素图标（如触手、古书、骰子等），统一采用描边风格
